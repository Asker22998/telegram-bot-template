# Telegram Bot Template 

A lightweight, fast and secure Telegram bot template built with **Python** and **aiogram 3**.  
Perfect as a starting point for your own bots without using paid builders or limited platforms.

---

## Features

- aiogram 3 (async, modern, fast)
- Secure token storage via `.env`
- Clean and scalable project structure
- Reply keyboard with button handlers
- Minimal dependencies
-  Easy to extend and customize
-  Ready for production and open-source

---

## Installation

```bash
1. clone the repository
git clone https://github.com/Asker22998/telegram-bot-template/
cd your-bot-template

2. install dependencies

pip install -r requirements.txt

3. change the token in .env

4. run the bot

python main.py
